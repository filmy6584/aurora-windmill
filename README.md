## 🛰️ What this app does

aurora-windmill is a desktop dashboard for planning Northern Lights photos in the Netherlands. It brings together live space weather, cloud data, and a detailed windmill map so you can pick a good spot before you head out.

You can use it to check:

- Live Kp index
- OVATION aurora chance
- DSCOVR solar wind data
- Cloud forecast
- Windmill locations on a map
- Light pollution over the map
- Photo planning for night sky sessions

## 🗺️ Using the dashboard

The main screen is built to help you decide where to go for a night sky photo.

### Check space weather first

Look at the Kp index and OVATION aurora probability. These give a quick view of how strong the aurora may be.

### Review cloud cover

Clouds matter a lot for night photography. A clear sky can make the difference between a useful trip and a wasted one.

### Pick a windmill location

The map shows Dutch windmills so you can find a foreground that fits an aurora photo.

### Watch light pollution

Light pollution can wash out faint aurora and stars. Use the overlay to find darker areas.

### Compare data in one place

The app puts weather, space weather, and map data together. That saves time when you are deciding where to go.

## 📍 Best way to use it for photo planning

If you want the best results, follow this order:

1. Check the aurora chance
2. Check the Kp index
3. Check solar wind data
4. Check cloud cover
5. Look for a windmill with low light pollution
6. Plan your trip for a dark and clear night

This helps you avoid poor viewing spots and gives you a better chance of getting a sharp image with a strong foreground.

## 🔄 Data shown in the app

The dashboard uses live or near-live sources to help with planning.

### Kp index

This gives a simple measure of geomagnetic activity. Higher values can mean a better chance to see aurora farther south.

### OVATION aurora probability

This shows the chance of aurora based on current space weather models.

### DSCOVR solar wind

Solar wind data helps show what is coming toward Earth from the Sun.

### Cloud forecasts

Cloud data helps you see if the sky will stay clear enough for photos.

### Light pollution overlay

This shows brighter areas and darker areas on the map so you can choose a better spot.
